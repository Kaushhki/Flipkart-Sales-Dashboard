# Flipkart Sales Performance Dashboard

# Project Overview
This project analyzes Flipkart sales data and builds an interactive dashboard using Power BI.

# Tools Used
Python, Pandas, Matplotlib, Power BI

# Features
Revenue trend analysis

Category and product performance

Payment method insights

KPI metrics such as revenue, orders, and average order value
